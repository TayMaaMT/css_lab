Both of the following containers have the same height and width set using the height and width properties. The first one has the default box-sizing value, and the second one has the box-sizing set to border-box.

The height and width of the first one increase when the padding is added. The height of the second one remains the same. Both their heights and widths are set to 330px by 300px respectively using the height and width properties.

I have a padding of 30px. My height is set to 330px. My total height is: 390px. My width is set to 300px. My total width is 460px.

To avoid that, make sure you don't set any fixed heights on your elements. It is almost always best to let the element's height adjust to the content inside it.

I have a padding, and my box-sizing property has been set to border-box. My height is adjusted to my content. My width is set to 300px. My total width remains 300px no matter how much padding is added.

I have more content than the second element, and the same padding, but my content does not get cut off because my height is set to auto, which means that my height increases as my content does.

The box-sizing property still has the same effect on my width.
